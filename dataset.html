About Dataset

Dataset for this project was obtained from GitHub. It can be downloaded through this link.

There is a total of 5634 rgb images of dimension 250 x 250 pixels, all of which are labelled. Each image is labelled to one of the 12 classes. The 12 possible categories are â€“ Apple Scab, Apple Black Rot, Apple Healthy, Grape Black Rot, Grape Esca, Grape Healthy, Grape Leaf Blight, Potato Early Blight, Potato Healthy, Potato Late Blight, Strawberry Healthy, Strawberry Leaf Scorch. These classes are some of the most common plant disease found in Apple, Grape, Potato or Strawberry.

Each class label has roughly between 420 to 550 images. It can be seen in the line chart below.



The Dataset is fairly balanced as seen in the bar chart below.



The pie chart below shows what proportion of dataset belong to each class label :



We used 80% of the dataset for Training, 10% for Validation and remaining 10% for Testing.
